# Règle : Stack Technique

## Frontend
- Next.js 14+ (App Router)
- Tailwind CSS + shadcn/ui
- Lucide React (icons)
- TypeScript strict

## Backend
- Next.js API routes
- Supabase (PostgreSQL + Auth + Storage)
- Zod (validation)

## Infrastructure
- Vercel (hosting)
- Stripe (paiements)

## Conventions
- Server Components par défaut
- Mobile-first
- Français dans UI
- 2 espaces indentation
